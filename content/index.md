## Philosophy on Obsidian

For those that have been in the 39th and have seen Nio's work in the Notion, you might question my decision to move to obsidian. The rationale behind it is tied directly to the philosophy of obsidian and the obsidian quartz digital garden. 

The philosophy behind obsidian is that it can be taken **offline**. You do not really need any internet connection to access all markdown files that can be found here. However, as obvious as it is, I think I still have to say it - you need to be **online** and **have a stable internet connection**. All you need to access these markdown files are <mark style="background: #D2B3FFA6;">git</mark> ( to pull the repository of where this is deployed from ), and <mark style="background: #D2B3FFA6;">Obsidian</mark> with the Git community plugin installed.

Please see related links should you want to access these offline.
